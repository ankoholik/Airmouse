# Описание датасета AirMouse

## Структура датасета

Датасет хранится в папках по классам:

```text
data/
  1/   # нет действия
  2/   # курсор
  3/   # левый клик
  4/   # правый клик
  5/   # прокрутка
```

Каждый файл — один пример в формате `.npy`.

### Сруктурированная раскладка (train/test + index)

Чтобы избежать огромных «плоских» директорий и сделать разбиение воспроизводимым, предпочтительно использовать:

```text
data/
  raw/
    <class_id>/<shard>/<n>.npy
  index.csv
  splits/train.csv
  splits/test.csv
```

## Объект, признаки, целевая переменная

- Объект: один сохранённый пример позы руки.
- Признаки: 63 значения типа float (`21 landmark * 3 координаты`).
- Цель: ID класса жеста (`1..5`).

## Скрипт сбора данных

Для сбора примеров с веб-камеры используйте `create_dataset.py`.

- Выберите ID класса в терминале.
- Нажмите `Space` в окне захвата, чтобы сохранить один пример.
- Примеры записываются в `datasets/<gesture_id>/`.
