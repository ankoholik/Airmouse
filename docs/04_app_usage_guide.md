# Гайд: как пользоваться AirMouse (GUI)

Этот документ описывает, как запустить и использовать desktop‑приложение AirMouse для управления курсором с помощью жестов руки.

## Быстрый старт

### Требования

- Windows 10/11
- Python 3.10+ (рекомендуется)
- Веб‑камера

### Установка зависимостей

Из корня проекта:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements-dev.txt
```

### Запуск приложения

Из корня проекта:

```bash
python -m airmouse
```

Альтернатива (тоже из корня проекта):

```bash
python airmouse/app.py
```

Откроется окно **AirMouse**.

## Как пользоваться приложением

### 1) Выбор камеры и настройка распознавания

Откройте вкладку **«Настройки»** и проверьте:

- **Камера**: выберите нужный индекс камеры (если их несколько).
- **Detection/Tracking confidence**: при необходимости увеличьте значения, если распознавание нестабильное.
- **Параметры мыши**: чувствительность, зона (ROI margin), шаг прокрутки, задержки/пороги клика.
- **Compute device**: выберите устройство вычислений (если доступно).

Нажмите **Apply** (в интерфейсе). Часть настроек применяется при следующем запуске видеопотока.

### 2) Запуск и остановка видеопотока

Слева в окне — видеопоток с камеры и кнопки управления:

- **Start**: запускает обработку камеры и распознавание жестов.
- **Stop**: останавливает обработку.

Если поток не запущен, в области видео будет текст **«Видео не запущено»**.

### 3) Просмотр распознаваемых жестов

Вкладка **«Жесты»** показывает список доступных жестов (как минимум — их названия).

Если вы добавляете/обновляете набор жестов, проверяйте, что нужные жесты появляются именно здесь.

### 4) Статистика и логи событий

На вкладке **«Статистика»** отображаются текущие метрики и (опционально) таблица событий распознавания.

Во вкладке **«Настройки»** можно включить/выключить логирование событий и задать интервал (это влияет на таблицу событий в статистике).

### 5) Графики

На вкладке **«Графики»** отображаются графики по собранной статистике (обновляются во время работы).

### 6) Экспорт отчетов

Во вкладке **«Отчеты»** можно экспортировать результаты:

- **PDF**
- **XLSX**
